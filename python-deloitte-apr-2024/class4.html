Review

While loops

The while loop syntax:


                    while <condition>:
                        <statement>
                        <statement>

As long as the condition is true, the statements below it are executed.


                    multiplier = 1
                    while multiplier <= 5:
                        print(9 * multiplier)
                        multiplier += 1

Infinite loops

Uh oh..

Make sure that your While loops condition has a way to become false


                    counter = 1
                    while counter < 5:
                        total += pow(2, counter)

The break statement

To prematurely exit a loop, use the break statement:


                    counter = 100
                    while counter < 200:
                        if counter % 7 == 0:
                            first_multiple = counter
                            break
                        counter += 1

While True Loop

This is a useful loop, just make sure you have a break, usually under an if statement


                    counter = 100
                    while True:
                        if counter % 62 == 0:
                            first_multiple = counter
                            break
                        counter += 1

⚠️ Be very sure that you're not coding an infinite loop!

String conversion

Concatenating strings with numbers

The following results in an error:


                        count = 3
                        noun = 'cat'
                        plural = 'I have ' + count + ' ' + noun + 's'

Strings can only be concatenated with other strings.
You can convert other values using str():


                    plural = 'I have ' + str(count) + ' ' + noun + 's'

This is called type casting

For loops

For loop

The for loop syntax:


                    for <variable_name> in <sequence>:
                        <statement>
                        <statement>

For Loops is for iterating over some sort of sequence.


                        for num in range(1, 5):
                            print(9 * num)

range()

If just one argument, range starts at 0 and ends just before it:


                        for num in range(6):
                            print(num)       # 0, 1, 2, 3, 4, 5

If two arguments, range starts at first and ends just before second:


                        for num in range(1, 6):
                            print(num)       # 1, 2, 3, 4, 5

If three arguments, the range steps at that rate:


                        for num in range(1, 6, 2):
                            print(num)       # 1, 3, 5

Lists

Lists

A list is a container that holds a sequence of related pieces of information.

The shortest list is an empty list, just 2 square brackets: 


                    members = []

Lists can hold any Python values, separated by commas:


                        members = ["Pamela", "Tinu", "Brenda", "Kaya"]

                        ages_of_kids = [1, 2, 7]

                        prices = [79.99, 49.99, 89.99]

                        remixed = ["Pamela", 7, 79.99]

List Index

Each list item has an index, starting from 0.


                    members = ["Pamela", "Tinu", "Brenda", "Kaya"]
                    # Index:      0         1        2       3

Accessing items (Bracket notation)


                    members = ["Pamela", "Tinu", "Brenda", "Kaya"]
                    # Index:      0         1        2       3

Access each item by putting the index in brackets:


                        members[0]  # Pamela
                        members[1]  # Tinu
                        members[2]  # Brenda
                        members[3]  # Kaya
                        members[4]  # 🚫 Error!

Negative indices are also possible:


                        members[-1]  # Kaya
                        members[-2]  # Brenda
                        members[-6]  # 🚫 Error!

Modifying list items

Python lists are mutable, which means you can change the values stored in them.

Use bracket notation to modify single items in the list:


                        members = ["Pamela", "Tinu", "Brenda", "Kaya"]
                        members[2] = "Jessica"
                        members[-1] = "Abby"

Try in PythonTutor

What if we change the value of an item beyond the list length?


                        members[5] = "Denise" # 🚫 Error!

Modifying lists with methods

Don't worry, you don't have to memorize all of theses

Method Description
.append(item) Adds item to the end of the list.
.insert(index, item) Inserts item at the specified index. This shifts all items after the specified index forward by one index.

.remove(item) Remove the first item from the list whose value is equal to item. It raises a ValueError if there is no such item.
.index(item) Returns the index of the first occurrence of an item whose value is equal to item.
.pop([index]) If no index is specified, it removes the last item from the list and returns it. Otherwise, it removes the item at the specified index and returns it.

See more in Python.org documentation

Examples of list modifying methods


                    groceries = ["apples", "bananas"]

                    groceries.append("peanut butter")
                    groceries.insert(0, "applesauce")
                    groceries.insert(3, "cheerios")
                    groceries.remove("bananas")

                    bought_food = groceries.pop()
                    bought_food2 = groceries.pop(1)
                    i = groceries.index("cheerios")
                    bought_food3 = groceries.pop(i)

Checking if value is in a list

The in keyword checks if a value is in a list (or any sequence).


                            if <item> in list:
                                <statement>
                        

                    if "bananas" in groceries:
                        print("⚠️ Watch your step!")

                    if "bleach" in groceries:
                        print("☣️ Watch what you eat!")

When used inside if conditions, in expressions return a Boolean (True/False).

Nested Lists

Nested Lists

Since Python lists can contain any values, an item can itself be a list.


                        gymnasts = [
                            ["Brittany", 9.15, 9.4, 9.3, 9.2],
                            ["Lea", 9, 8.8, 9.1, 9.5],
                            ["Maya", 9.2, 8.7, 9.2, 8.8]
                        ]
  • What's the length of gymnasts? 3
  • What's the length of gymnasts[0]? 5
Can also think of it in terms of rows and columns.

Dynamic creation of nested lists

If we start with...


                            gymnasts = [
                                ["Brittany", 9.15, 9.4, 9.3, 9.2],
                                ["Lea", 9, 8.8, 9.1, 9.5]
                            ]

We can append another list a few ways:


                    gymnasts.append(["Maya", 9.2, 8.7, 9.2, 8.8])
                    

                    gymnasts += ["Maya", 9.2, 8.7, 9.2, 8.8]
                    

                        gymnast = []
                        gymnast.append("Maya")
                        gymnast.append(9.2)
                        gymnast.append(8.7)
                        gymnast.append(9.2)
                        gymnast.append(8.8)
                        gymnasts.append(gymnast)

Accessing nested list items


                        gymnasts = [
                            ["Brittany", 9.15, 9.4, 9.3, 9.2],
                            ["Lea", 9, 8.8, 9.1, 9.5],
                            ["Maya", 9.2, 8.7, 9.2, 8.8]
                        ]

Access using bracket notation, with more brackets as needed:


                    gymnasts[0]    # ["Brittany", 9.15, 9.4, 9.3, 9.2]
                    gymnasts[0][0] # "Brittany"
                    gymnasts[1][0] # "Lea"
                    gymnasts[1][4] # 9.5
                    gymnasts[1][5] # 🚫 IndexError!
                    gymnasts[3][0] # 🚫 IndexError!

Modifying nested lists


                    gymnasts = [
